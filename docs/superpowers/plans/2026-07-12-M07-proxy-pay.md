# M07 找人代付 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans.  
> **Status:** ✅ 已实现（2026-07-12，commit `ce91df3` + review fix `40db545`）

**Goal:** 代付链接生成、公开预览、登录后代付人发起 JSAPI/fake 支付，回调写入 `paid_by_user_id`，Frontend H5 落地页。

**Architecture:** 独立 `ProxyPay` 限界上下文；复用 M06 网关与 `ConfirmPaymentHandler`；Token 有效期绑定订单 `auto_cancel_minutes`。

**Tech Stack:** PHP **8.4** · Laravel **12** · React **18.3** · Vite **5.x**

**Spec:** [2026-07-12-M07-proxy-pay-design.md](../specs/2026-07-12-M07-proxy-pay-design.md)

## Global Constraints

- 依赖 M05 订单 `payment_method=proxy`、M06 支付网关
- API 前缀 `/api/v1/`
- DDD 分层 + Feature 测试
- H5 路由在 frontend 公开区域（不经 AdminLayout）
- `FRONTEND_URL` 配置项用于生成分享链接
